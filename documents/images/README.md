# documents/images

Visual assets for the TFG thesis. Each subdirectory maps to a chapter (and aligns with the LaTeX figure paths used in the memoria source).

## Layout

- `05_results/` — chapter 5 (results) figures. Generated by `notebooks/agents/N33_thresholds_and_calibration.ipynb`.

## Files in 05_results/

- `threshold_sweep_overtake.png` — Section A of N33
- `threshold_sweep_sc.png` — Section B of N33
- `threshold_sweep_undercut.png` — Section C of N33
- `mc_dropout_coverage.png` — Section D of N33

All four PNGs are written at 300 DPI with `bbox_inches="tight"` and saved by N33 in a single Run All pass.
