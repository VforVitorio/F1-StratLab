# data/eval

This directory holds benchmark output artefacts referenced by chapter 5 of the TFG thesis. Each entry has a CSV (dot decimal, pandas-loadable, ready for `df.to_latex`) and a Markdown table (comma decimal, ready to embed in LaTeX). All files are regenerable from the listed source; do not edit them by hand.

## Provenance table

| Artefact stem               | Generated by                                                            | Notes                                                                                |
|-----------------------------|-------------------------------------------------------------------------|--------------------------------------------------------------------------------------|
| `pace_baselines`            | `scripts/bench_pace_baselines.py`                                       | persistence + team x circuit median + XGBoost delta on 2025 holdout                  |
| `whisper_results`           | `scripts/bench_whisper.py`                                              | Whisper turbo per-clip latency on CUDA, WER not yet measured                         |
| `subagent_latency`          | `scripts/bench_subagent_latency.py`                                     | six sub-agents in isolation on Suzuka 2025 NOR lap 21                                |
| `nlp_pipeline_cpu`          | `scripts/bench_nlp_pipeline_cpu.py`                                     | sentiment + intent + NER on CPU and GPU                                              |
| `threshold_sweep_overtake`  | `notebooks/agents/N33_thresholds_and_calibration.ipynb` (Section A)     | overtake probability threshold sweep                                                 |
| `threshold_sweep_sc`        | `notebooks/agents/N33_thresholds_and_calibration.ipynb` (Section B)     | safety car probability threshold sweep                                               |
| `threshold_sweep_undercut`  | `notebooks/agents/N33_thresholds_and_calibration.ipynb` (Section C)     | undercut success threshold sweep                                                     |
| `mc_dropout_coverage`       | `notebooks/agents/N33_thresholds_and_calibration.ipynb` (Section D)     | MC dropout interval coverage for tire degradation predictions                        |

## How to regenerate

Run the four benchmark scripts:

```bash
uv run python scripts/bench_pace_baselines.py
uv run python scripts/bench_whisper.py
uv run python scripts/bench_subagent_latency.py
uv run python scripts/bench_nlp_pipeline_cpu.py
```

Then open each notebook and use Restart Kernel -> Run All:

- `notebooks/agents/N33_thresholds_and_calibration.ipynb` — produces all four `threshold_sweep_*` and `mc_dropout_coverage` artefact pairs.
